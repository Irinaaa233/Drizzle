# Drizzle Brand Website

A minimalist and elegant brand website for Drizzle, featuring a clean design with a warm cream color palette (#fffdf3).

## Features

- **Modern Design**: Clean, minimalist interface with smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Scrolling**: Enhanced navigation with smooth scroll behavior
- **Interactive Elements**: Hover effects and fade-in animations
- **Contact Form**: Functional contact form (frontend only)

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality and animations
- `README.md` - This file

## Getting Started

Simply open `index.html` in your web browser to view the website. No build process or dependencies required.

## Sections

1. **Hero Section** - Main landing area with brand introduction
2. **About Section** - Brand story and statistics
3. **Products Section** - Product categories showcase
4. **Contact Section** - Contact form and information
5. **Footer** - Copyright and branding

## Customization

The website uses CSS custom properties (variables) defined in `styles.css` for easy customization:

- `--primary-color`: Main background color (#fffdf3)
- `--text-primary`: Primary text color
- `--accent-color`: Accent color for buttons and highlights
- `--border-color`: Border and divider colors
